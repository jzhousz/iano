Comparison mode allows us to build chains of algorithms. Each chain can contain multiple feature extractors which is different from the single mode. We can make many chains and select all of them and run them in sequence. This will then bring us to the result window which shows the results of each chain.

Button Explaination
New Create a new, empty chain.
Remove Remove the currently highlighted chain. This does not remove any chains that have their check boxes checked. To remove the current feature selectors or extractors, add "None" to the chain.
Save Chains Save all the created chains (the chains currently in the table).
Load Chains Load previously created chains that were stored in a chain file. The extension for that file would be ".ichn".
Run Run the chains that have been checked. You need to have at least one chain checked for the button to be enabled.
Save Model Save the current model. If there are multiple chains compared, the one with the highest accuracy is saved.
Annotate Go into annotation mode with the current model.
Tips Open up this window.
Copy Selected Create a copy of the currently highlighted chain.
Stop Stop the running of the current chain(s). It will finish up the current operation and then stop rather than stopping in the middle of it.
